使用pyinstaller打包py文件成exe程序

将cmd的目录切换至（命令：cd 文件路径(注意空格)）需要打包的py文件目录下：

pyinstaller -F test.py

常用参数说明：
–icon=图标路径
-F 打包成一个exe文件
-w 使用窗口，无控制台
-c 使用控制台，无窗口
-D 创建一个目录，里面包含exe以及其他一些依赖性文件
pyinstaller -h 来查看参数
